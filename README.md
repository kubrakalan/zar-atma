# Zar Atma Uygulaması

Bu proje, basit bir web tabanlı zar atma uygulamasıdır. Kullanıcılar iki zar atabilir, rastgele sayılar üretebilir ve toplam sonucu görebilir. Animasyon ve ses efekti ile eğlenceli bir deneyim sunar.

## Özellikler

- İki zar ile rastgele sayı üretimi (1-6 arası)
- Zar atma animasyonu (sallanma efekti)
- Ses efekti (zar atma sesi)
- Responsive tasarım (mobil uyumlu)
- Kolay kullanım: Tek butonla zar atma

## Nasıl Çalıştırılır

1. Bu repo'yu klonlayın:
   ```
   git clone https://github.com/kubrakalan/zar-atma.git
   ```

2. Proje klasörüne gidin:
   ```
   cd zar-atma
   ```

3. `index.html` dosyasını bir web tarayıcısında açın (örneğin, çift tıklayın veya live server kullanın).

Alternatif olarak, VS Code'da Live Server extension ile çalıştırabilirsiniz.

## Teknoloji Yığını

- **HTML5**: Sayfa yapısı
- **CSS3**: Stil ve animasyonlar
- **JavaScript (ES6)**: Etkileşim ve mantık
- **Audio API**: Ses efekti

## Dosya Yapısı

```
zar-atma/
├── index.html          # Ana HTML dosyası
├── css/
│   └── style.css       # Stil dosyası
├── js/
│   └── script.js       # JavaScript mantığı
└── sounds/
    └── dice.mp3        # Zar atma ses efekti
```
